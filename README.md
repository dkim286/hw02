# HW02: Processes & Threads
Programming assignment for CSUF's Operating Systems class. Developed in conjunction with [Chris](https://github.com/cdnutter).

### Building

#### `sorting.c`

``` shell
$ make sorting
```

#### `time_pipe.c`

```shell
$ make pipe
```

#### `time_shm.c`

```shell
$ make shm
```

### Usage

```shell
$ ./sorting
```

```shell
$ ./time_pipe sleep 3
```

```shell
$ ./time_shm ls -la
```
